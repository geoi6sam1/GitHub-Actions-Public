* **部署账号密码步数 [`index.py`](https://github.com/geoi6sam1/GitHub-Actions-Public/blob/main/huami-step/index.py) 如下：**

```python
# 帐号（这里只能是邮箱）
user = "email@huami.step"

# 步数范围（尽可能随机）
step = str(randint(17760,19999))
# 钉钉消息（发送：1，不发送：其他字符）
dingding = 1
```
